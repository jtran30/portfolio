Our project is based off on Andrew Johnson's work on: https://www.evl.uic.edu/aej/491/ Our general approach to this project was to have something scalable whilst still being able to witness proportions of our solar system vs. other systems. Clearly our star, the sun, is outmatched by many other stars such as the 11 COM, which is shown at the start at running our project.

At the beginning at our project, the user should immediately notice that there's a menu in front of the user, and is encouraged to mess with the settings seen on this menu.

The user will eventually find out that they can toggle this menu using the VIVE Controller (left). On the menu, there are multiple options that can help assist the user scale properties of the solar systems seen, such as the orbit/time speed, planet size, and orbit size. The user should also then see 4 3d systems, with multiple (40) 2d systems shown in the back.

With the user having the VIVE controller, they should test out that clicking onto them with a laser should switch out the first solar system (our sun), with the system they clicked on. Should the user bring the menu up again, they can cycle through pages (600 systems total) of 2d systems, and then can replace 1,2,3, or 4th system by selecting the system to modify. Users should also hear a clicking feedback, as well as some ambient sound through each star.
